# Prompt Canvas

React + Vite + Tailwind App zum Generieren und Rendern von UI-Funktionen via Groq.

## Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Funktionen

- Groq API Key im Start-Modal
- Fullscreen-UI
- Double Click: Prompt-Text eingeben
- Hold: Speech Recognition starten
- Swipe Left: History
- Swipe Right: Chats
- KI-Ausgabe wird als JavaScript-Funktion ausgeführt und in das Output-Element gerendert
